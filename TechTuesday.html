Topic: Artificial Intelligence March 19 2024

Black Box

An AI black box is an AI system with software that is invisible to the user. A user can give an input and receive an output, but cannot see the internal workings of the system's code or logic. AI uses something called machine learning, which uses algorithms, training data and a model. It learns how to identify patterns after being trained with an extensive set of examples. Once the algorithm has been trained, the result is a machine learning model. This model is what is used for AI programs such as Chat GPT and DALL-E2. For example, these programs would be able to identify patterns, like finding a specific object in an image. You would feed the algorithm images and get an output on whether they have that object or not. In a black box any of the components that make up machine learning can be hidden, but in many cases the algorithm is already known, defeating the purpose of having the black box. So in many cases the training data would be put into the black box to stop others from seeing the information.

Black boxes may appear to be a good idea, however they can result in more harm than good. These algorithms are not transparent with users and there is no way for people to know the machine’s reasoning behind the decisions it makes. This could cause issues for cases like asking an AI to give a medical diagnosis, because it would be unable to give a reason for its decision. The reason black boxes were created was to stop hackers from accessing the code, but this had been revealed to not completely work because of the option of reverse engineering software. In conclusion, black boxes are not all that helpful when it comes to AI programs that impact people.

Topic: Hardware February 27 2024

Motherboard

A motherboard is a piece of internal hardware. It distributes electricity and controls communication between the CPU and RAM. There are many types of motherboards that are each compatible with specific computers depending on the size, processors, and memory whereas they can all work with a variety of hard drives. On the earlier versions of motherboards there were fewer components, they usually had a CPU and card ports, but users had to add things like memory and controllers for floppy drives like on the first IBM PC motherboard.

When a computer is used, electricity gets transferred to the motherboard. Data is then moved between the chipset components and travels between the southbridge and northbridge sections, which interact with other parts of the computer such as CPU, RAM, PCI and PCIe through electrical signals that pass through a microchip to either bridge. The information sent will be encoded through binary and translated in a split second. The northbridge helps data connections to the RAM, operations from the RAM are interpreted by the CPU and acts as the output after the RAM delivers the inputs. After information is sent to the PCIe, the data is copied or moved to an expansion card, depending on the card being used. The southbridge controls data connection to the BIOS, USB and the SATA. Your computer turns on because of the signals going to the BIOS and the data going to the SATA, which turns on your optical, hard disc and solid state drives. The information saved on the SATA gives power to many things such as the video card, sound card and the network card. Motherboards allow the different aspects of the computer to communicate, and are an essential part of making the computer work.

Topic: Computers February 20 2024

Raspberry Pi

Raspberry Pis were created in 2012 in the UK. Though computers are widely used devices, in many developing countries they are hard to access, which is where the raspberry pie comes in. They are a low costing single computer board around the size of a credit card. They were created to be widely accessible while still being affordable and useful. Raspberry Pi’s are mini-computers that are able to exchange and process information from multiple different devices. They are often used to learn programming or for data management. A Raspberry Pi has similar features to a motherboard while taking out things that are not necessary, such as its internal storage. To use a Raspberry Pi you will need an SD card containing an operating system, which is needed for the computer to work. Raspberry Pis are compatible with the Linux OS which reduces the amount of storage required. After setting up the OS to the Raspberry Pi, you can connect the Raspberry Pi to a monitor, HDMIs, television, mouse, or keyboard as an output device.

Raspberry Pis have many features that make them a popular and very versatile tool. These include: CPU, HDMI and USB ports, GPU, Memory, an Ethernet port, an SD card, and GPIO pins. The CPU (Central Processing Unit) acts as the main base of operations for the Raspberry Pi and is used to perform actions using logical and mathematical operations. An HDMI (High Definition Multimedia Interface) is useful because it gives options for video output, and it has a USB port for connecting to other devices. The GPU (Graphic Processing Unit) helps the Raspberry Pi to make quicker calculations. It has memory that stores real-time information to be easily referred back to later. It has an Ethernet port for wired internet access. An SD (secure digital) card is used instead as a storage device and is necessary for the Raspberry Pi to function, as it contains the operating system that turns it on and stores data. GPIO (General Purpose Input and Output) pins can be used to interact with other electronic circuits and can be programmed to read and control electronic signals from other boards. Finally, it has LEDs (Light Emitting Diodes) with different colors to show different things, like if it is powered on or to monitor Ethernet connection.
